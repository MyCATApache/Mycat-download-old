MyCat 1.2

Changelog

MyCat 1.2 (svn:r478)
特性
+ 增加了SQL兼容的路由解析特性，通过SQL注解方式
+ 增加了数据库方式的全局序列号的实现，可以用于集群模式
+ 增加了查询SQL默认限制分页数的特性，如select * from orders会默认改写为select * from orders limit 100,其中100是在schema上进行配置的参数， schema name="TESTDB" checkSQLschema="true" sqlMaxLimit="100"
+ 实现了多个写节点的模式，随机选取一个写节点，支持percona cluster
+ 实现了AIO并优化了线程池设计，目前AIO只采用一个线程池
+ 可以配置绑定本地的IP地址
Fix bug
* 修复AIO连接僵死的BUG
* 前端连接的writeQueue改为LinkeQueue，防止队列满导致线程阻塞
* AIO优化
* 修复了夸分片排序情况下字符串和日期的排序问题
* 修复了读写分离的缺陷（双主模式以及读取主节点的问题）
* 修复了不跨分片的SQL分页LIMIT值被修改的缺陷
* 修复了跨分片SQL异常时候连接被关闭的问题
